# Getting Started

## Prerequisites

- Dotnet Sdk Installed

## Dev Usage

```bash
  dotnet run
```

## Test

```bash
  yarn run test:coverage
```

## Publish - SANDBOX

```bash
   dotnet publish -c Release -o ./publish
```

## Public - DEV

```bash
   dotnet publish App.dev.csproj -c Release -o ./publish
```
